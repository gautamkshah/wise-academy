# WISEACADEMY

Scalable educational platform for coding students.

## Project Structure

- `backend/`: NestJS application (API, Auth, CRON jobs)
- `frontend/`: Next.js application (UI, Dashboard, Course Player)

## Getting Started

### Backend

```bash
cd backend
npm install
npm run start:dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
